# top

Shows information about running tasks.

**Syntax:** ```top[]```

```o
o)top[]
tid handle       name                      state   created      run          iowait       total        load
-----------------------------------------------------------------------------------------------------------
0   <Reagent#3> "main"                    IOWait  21:53:20.172 00:00:00.336 00:00:00.109 00:00:00.445 0
o)
```
