# top

Shows information about running tasks.

**Syntax:** ```top[]```

```o
o)top[]
tid handle      name   state   created      run          iowait       total        load
---------------------------------------------------------------------------------------
0   <Reagent#4> "main" Running 14:46:25.711 00:00:01.297 00:00:00.000 00:00:01.297 100
o)
```
