# top

Shows information about running tasks.

**Syntax:** ```top[]```

```o
o)top[]
tid handle       name                      state   created      run          iowait       total        load
-----------------------------------------------------------------------------------------------------------
0   &ltReagent#3&gt "main"                    IOWait  15:15:21.371 00:00:00.256 00:00:05.998 00:00:06.254 0
o)
```
