# println

Prints down a string line representation of `x`. Format string contains `%` at each position to be replaced by formatted v–∞lue.

**Syntax:** ```println[".. %";x..]```

```o
o)println["The final result is %";42]
The final result is 42
o)println["a is %, b is %";1;2 3 4 5]
a is 1, b is 2 3 4 5
o)
```
