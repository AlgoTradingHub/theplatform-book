# Monadic !

When applied to type spec returns corresponding internal type id.

**Syntax:** ```!x; ![x]```

```o
o)!`s`int
64
o)!`v`int
45312
o)!`v`long
45376
o)!``dict
126992
o)
```

See the full list of type specs [here](/reference/types/scalars/scalars.md).
